# ContributionGuidelines

## Git branch names
---
Categorize by a slash(/).

For example,
```
feat/new-feature

feat/i-am-hungry

refactor/rename-variables

fix/wrong-type-declarations
```


## Git commit messages
---
```
commit-type: what you did.
```

**Available types**
- feat
- fix
- refactor
- chore
- docs
- style
- test
- etc...
